# Mirage VPN - حماية سحابية لجميع الفئات

تطبيق في بي إن (VPN) مخصص يجمع بين الأداء الخارق للألعاب بفضل المسارات السحابية الخاصة (Cloud Routes) والتسليح الرقمي لجميع الفئات وتوفير خطط متنوعة واشتراكات ومكافآت الإعلانات.

## المميزات
- **نظام التمويه والتشفير المتقدم**: يدعم VLESS/Reality و Gzip والتخفي.
- **مسار الألعاب السحابي**: توجيه خاص للتطبيقات المُختارة للحفاظ على بينغ (Ping) منخفض جداً ودقة استجابة عالية.
- **تدمير البيانات ذاتياً (Self-Wipe)**: يمحو إعدادات التكوين واللوغز (Logs) بمجرد سحب التطبيق.
- **دعم شامل للمشتركين والفئات المجانية**: عن طريق مشاهدة إعلانات الفيديو لمكافآت الوصول السريع أو الاشتراكات.
- **واجهات احترافية**: تصميم نيون (Neon Glassmorphism) متقدم يمنح إحساساً بالقوة.

## ملاحظات الأمان
لا تضع روابط السيرفرات الحقيقية في الكود المرفوع.
استخدم أدوات التوقيع المناسبة (KeyStore Security Signature) عند التجميع (Build) للنسخة الإنتاجية.

## التقنيات المستخدمة
- Kotlin & Jetpack Compose
- Xray-core / V2Ray / VLESS
- Google AdMob (Rewarded Videos)
- Google Play Billing Library
- Firebase & GCP (Cloud Sync)
