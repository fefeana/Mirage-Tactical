Run Locally

Prerequisites: Node.js  

1. Install dependencies:  
   `bash
   npm install
   `
  2. Set the GEMINIAPIKEY in .env.local to your Gemini API key  
3. Run the app:  
   `bash
   npm run dev
   `

Deployment 🚀

Deploy on GitHub Pages
1. تأكد أن الفرع الأساسي هو main.  
2. أضف ملف Auth.html في الجذر.  
3. من إعدادات المستودع → Pages → اختر الفرع main والمسار /root.  
4. الرابط سيكون:  
   `
   https://<username>.github.io/Mirage-Tactical/Auth.html
   `

Deploy on Firebase Hosting
1. ثبت Firebase CLI:  
   `bash
   npm install -g firebase-tools
   `
  2. سجل الدخول:  
   `bash
   firebase login
   `
  3. أنشئ مشروع جديد أو اربط بمشروع موجود:  
   `bash
   firebase init
   `
  4. ضع ملف Auth.html في مجلد public.  
5. انشر:  
   `bash
   firebase deploy
   `
